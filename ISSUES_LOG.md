## 2026-06-23 - 김단순 사용자 닉네임 변경 반영 오류 및 조치
- **발생 이슈**: 
  - 김단순 사용자의 요청에 따라 닉네임을 `Dandoon_Hyeyoung`에서 `Dansoon_Hyeyoung`으로 수동 변경하기 위해 Firebase Client SDK를 사용하여 업데이트를 시도했습니다.
  - 실행 결과 `PERMISSION_DENIED` 권한 거부 오류가 발생하여 변경 작업이 차단되었습니다.
- **원인 분석**:
  - Firestore 보안 규칙이 클라이언트 SDK에서 로그인 정보가 없는 익명 상태의 쓰기/수정 동작을 차단하고 있었기 때문입니다.
- **해결 방법**:
  - 프로젝트 루트에 보관된 Firebase 서비스 어카운트 키 파일을 활용하여 `firebase-admin` 라이브러리를 통해 Admin SDK 권한으로 안전하게 업데이트하는 스크립트를 작성하고 실행했습니다.
  - 최종적으로 김단순 사용자의 닉네임이 `Dansoon_Hyeyoung`으로 무사히 수정 완료되었습니다.
