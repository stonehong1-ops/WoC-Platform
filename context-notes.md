# 알림 내 TODO 작업 컨텍스트 및 결정 사항 노트

## 아키텍처 및 디자인 결정 사항
- **TODO 알림 전면 제거**: 과거 배지 카운트에서 제외된 것에서 나아가, 알림 전체 모델과 서비스 로직 및 UI 코드 상의 `'TODO'` 분류를 완전히 도려내어 알림 시스템을 일반 알림(`INFO`) 중심으로 단순화하고 가독성을 높입니다.
- **성역 불변성**: `tailwind.config.ts` 파일은 절대 손대지 않고 기존 Stitch 팔레트와 테일윈드 인라인 클래스를 활용합니다.
- **다국어(i18n) 동시 반영**: 작업 간 문구 수정 또는 번역 키 추가 시 한국어와 영어 사전에 동시에 누락 없이 적용합니다.
